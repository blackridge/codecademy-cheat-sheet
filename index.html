HTML TAGS REFERENCE

Basic Tags

Tag Name Description
<html> The HTML Document/ Root element Represents the root (top-level element) of an HTML document.
<head> The Document Metadata (Header) element Contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.
<title> The Document Title element Defines the document's title that is shown in a browser's title bar or a page's tab.
<body> The Document Body element Represents the content of an HTML document. There can be only one <body> element in a document.
<header> The Header element Represents introductory content, typically a group of introductory or navigational aids.
<footer> The Footer element Represents a footer for its nearest ancestor sectioning content or sectioning root element.
<article> The Article Content element Represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable (e.g., in syndication).
<section> The Generic Section Represents a generic standalone section of a document.
<p> The Paragraph Represents a paragraph. Usually represented in blocks.
<div> The Content Division element The generic container for flow content.
<span> The Content Span element A generic inline container for phrasing content, which does not inherently represent anything.
<img> The Image Embed element Embeds an image into the content.
<aside> THe Aside element Represents a portion of a document whose content is only indirectly related to the document's main content. Frequently presented as sidebars or call-outs.
<audio> The Embed Audio element Embed sound content in documents.
<canvas> The Graphics Canvas element Use either with the canvas script API or the WebGL API to draw graphics and animations.
<datalist> The HTML Data List element The HTML contains a set of <option> elements that represents the permissible or recommended options available to choose from within other controls.
<details> The Details disclosure element Creates a disclosure widget in which informaiton is visible only when the widget is toggled into an "open" state.
<embed> The Embed External Content element Embeds external content at the specified point in the document.
<nav> The Navigation Section element Represents a section of a page whoe purpose is to provide navigation links, either within the current document or to other documents.
<search> The generic search element A container representing the parts of the document or application with form controls or other content related to performing a search or filtering operation.
<output> The Output element A container element into which a site or app can inject the results of caculation or the outcome of a user action.
<progress> The Progress Indicator element Displaya an indicator showing the completion progress of a task, typically displayed as a progress bar.
<video> The Video Embed element Embeds a media player which supports video playback into the document.
<ul> The Unordered List element Represents an unordered list of items, typically rendered as a bulleted list.
<ol> The Ordered List element Represents an ordered list of items - typically rendered as a numbered list.
<li> The List Item element Represents an item in a list. It must be contained in a parent element: an ordered list, an ordered list, or a menu.
